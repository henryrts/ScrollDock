# Changelog

## 0.1.0 — 2026-07-22

- Added accessibility onboarding and prominent privacy disclosure.
- Added selected-app allowlist with ChatGPT selected by default.
- Added draggable, collapsible Top, Page Up, Page Down, and Bottom overlay controls.
- Added semantic accessibility scrolling with gesture fallback.
- Added hold-to-scroll, cancellation, edge feedback, and safety timeouts.
- Added portrait and landscape position persistence, size, opacity, page-distance, and timing settings.
- Added local navigation test screen, privacy documentation, unit tests, and Android build workflow.
